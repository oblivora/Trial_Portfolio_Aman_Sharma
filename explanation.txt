HTML, or HyperText Markup Language, is the standard language for creating web pages. It uses a system of **tags** to structure content. Each tag serves a specific purpose, telling the browser how to display the information.

Here's a basic format of an HTML document with explanations for common syntax:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
    <link rel="stylesheet" href="styles.css">
    <script src="script.js"></script>
    <style>
        /* CSS styles can go here */
        body {
            font-family: Arial, sans-serif;
        }
    </style>
</head>
<body>
    <header>
        <h1>Welcome to My Website</h1>
        <nav>
            <ul>
                <li><a href="#home">Home</a></li>
                <li><a href="#about">About</a></li>
                <li><a href="#contact">Contact</a></li>
            </ul>
        </nav>
    </header>

    <main>
        <section id="home">
            <h2>About Us</h2>
            <p>This is a paragraph of text. You can put your content here.</p>
            <img src="image.jpg" alt="Description of image">
            <a href="https://www.example.com" target="_blank">Visit Example.com</a>
        </section>

        <section id="products">
            <h2>Our Products</h2>
            <table>
                <thead>
                    <tr>
                        <th>Product Name</th>
                        <th>Price</th>
                    </tr>
                </thead>
                <tbody>
                    <tr>
                        <td>Product A</td>
                        <td>$10.00</td>
                    </tr>
                    <tr>
                        <td>Product B</td>
                        <td>$20.00</td>
                    </tr>
                </tbody>
            </table>
        </section>

        <section id="contact">
            <h2>Contact Us</h2>
            <form action="/submit-form" method="post">
                <label for="name">Name:</label><br>
                <input type="text" id="name" name="name" required><br><br>

                <label for="email">Email:</label><br>
                <input type="email" id="email" name="email"><br><br>

                <label for="message">Message:</label><br>
                <textarea id="message" name="message" rows="4" cols="50"></textarea><br><br>

                <input type="submit" value="Submit">
            </form>
        </section>
    </main>

    <footer>
        <p>&copy; 2025 My Website. All rights reserved.</p>
    </footer>
</body>
</html>
```

-----

### Explanation of HTML Syntax

Here's a breakdown of the common HTML tags and what they do:

  * **`<!DOCTYPE html>`**
    This declaration defines the document type and version of HTML being used (in this case, HTML5). It's always the first line in an HTML document.

  * **`<html lang="en">...</html>`**
    This is the **root element** of an HTML page. All other HTML content goes inside this tag. The `lang="en"` attribute specifies the language of the document, which helps search engines and screen readers.

  * **`<head>...</head>`**
    This section contains **meta-information about the HTML document**, which is not directly displayed on the web page. This includes:

      * **`<meta charset="UTF-8">`**: Specifies the character encoding for the document, ensuring proper display of various characters.
      * **`<meta name="viewport" content="width=device-width, initial-scale=1.0">`**: Configures the viewport for responsive web design, making the page adapt to different screen sizes.
      * **`<title>Document Title</title>`**: Sets the **title of the web page**, which appears in the browser tab or window title bar.
      * **`<link rel="stylesheet" href="styles.css">`**: Links an **external CSS stylesheet** (`styles.css`) to the HTML document, used for styling the page's appearance.
      * **`<script src="script.js"></script>`**: Links an **external JavaScript file** (`script.js`), used for adding interactive behavior to the page.
      * **`<style>...</style>`**: Allows you to embed **internal CSS styles** directly within the HTML document.

  * **`<body>...</body>`**
    This section contains all the **visible content of the HTML document**, which is displayed on the web page.

  * **`<header>...</header>`**
    Represents **introductory content or a set of navigational links**, typically found at the top of a page or section.

  * **`<h1>...</h1>` to `<h6>...</h6>`**
    These tags are used for **headings**. `<h1>` is the most important (main heading), and `<h6>` is the least important (smallest heading).

  * **`<nav>...</nav>`**
    Defines a section containing **navigation links**, typically for the main navigation of a website.

  * **`<ul>...</ul>`**
    Creates an **unordered list** (bullet points).

  * **`<li>...</li>`**
    Represents a **list item** within an ordered (`<ol>`) or unordered (`<ul>`) list.

  * **`<a href="#home">Home</a>`**
    Creates a **hyperlink**.

      * `href`: Specifies the destination URL (can be an external link, internal link, or anchor to a section on the same page).
      * `target="_blank"`: Opens the linked document in a new window or tab.

  * **`<main>...</main>`**
    Represents the **dominant content** of the `<body>`. It should be unique to that document and not contain content repeated across a set of documents (like sidebars, navigation links, copyright information, etc.).

  * **`<section>...</section>`**
    A **standalone section** within an HTML document, often with a heading. It groups related content.

  * **`<p>...</p>`**
    Defines a **paragraph of text**.

  * **`<img>`**
    Used to **embed an image** into the HTML document.

      * `src`: Specifies the path to the image file.
      * `alt`: Provides **alternative text** for the image, displayed if the image cannot be loaded or for screen readers, improving accessibility.

  * **`<table>...</table>`**
    Creates a **table** to display tabular data.

  * **`<thead>...</thead>`**
    Groups the **header content** of a table.

  * **`<tbody>...</tbody>`**
    Groups the **body content** of a table.

  * **`<tr>...</tr>`**
    Defines a **table row**.

  * **`<th>...</th>`**
    Defines a **table header cell**. By default, text inside `<th>` is bold and centered.

  * **`<td>...</td>`**
    Defines a **standard data cell** in a table.

  * **`<form>...</form>`**
    Creates an **HTML form** for user input.

      * `action`: Specifies where to send the form data when the form is submitted (e.g., a server-side script).
      * `method`: Specifies the HTTP method to use when sending form data (e.g., `get` or `post`).

  * **`<label for="name">Name:</label>`**
    Defines a **label for an input element**. The `for` attribute should match the `id` of the input it labels for accessibility.

  * **`<input>`**
    Creates various **input fields** for forms.

      * `type="text"`: A single-line text input field.
      * `type="email"`: An input field for email addresses.
      * `type="submit"`: A button that submits the form.
      * `id`: A unique identifier for the input element.
      * `name`: The name of the input field, used when submitting form data.
      * `required`: Makes the input field mandatory.

  * **`<textarea>...</textarea>`**
    Defines a **multi-line text input control** (a text area).

      * `rows`: Specifies the visible number of lines.
      * `cols`: Specifies the visible width in characters.

  * **`<footer>...</footer>`**
    Represents a **footer for its nearest sectioning content or for the entire document**. Typically contains copyright information, contact details, etc.

  * **\`\`**
    This is an **HTML comment**. Content inside these tags is ignored by the browser and is used by developers for notes or explanations.

-----

This covers the most common and fundamental HTML tags and their uses. HTML is constantly evolving, but these elements form the core of almost every web page you'll encounter.
